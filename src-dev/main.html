Introduction

A system designed to find the name of movie/tv episode where a phrase or a list of phrases occured.

Features

You can query for the episode where a list of phrases occur by entering the phrases in query box and our system will output top three possible candidates for the query

You can also update our system by providing transcript of any movie/tv episode alongwith the name of the episode in update tab. If the episode doesn't exists in our system then it'll be automatically added, archived, and updated.
